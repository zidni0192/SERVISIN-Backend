## Clone or download
- `$ git clone https://github.com/zidni0192/SERVISIN-Backend.git`
- `$ cd SERVISIN-Backend`
- `$ npm install`
## Create Environment Variable
```
$ touch .env
$ nano .env
```

```
DB_HOST="Your_Host"
DB_USER="Your_Username"
DB_PASSWORD="Your_Password"
DB_NAME="Your_Table"

SERVER_PORT=9999
```
## Start Application
`$ npm start`
